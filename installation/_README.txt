This folder is for stuff related to installation of SLiM/SLiMgui.  At present, it contains:

Ubuntu_18.04_GNOME.txt : instructions for integrating SLiMgui into GNOME on Ubuntu 18.04 LTS

DebianUbuntuInstall.sh : a Bash script to build and install SLiM and SLiMgui on Debian-based Linux systems, such as Ubuntu.
